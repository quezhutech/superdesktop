## ✨ 核心功能

### 智能思维导图
输入一句话，自动生成结构化思维导图，支持导出图片和实时编辑，让创意瞬间可视化。

### 极速灵感笔记
侧边栏随手记，支持 Markdown 语法，自动关联当前对话上下文，再也不会遗失任何闪现的灵感。

将复杂的项目拆解为可执行的任务清单，AI 自动预测完成时间，助您高效掌控工作节奏。
